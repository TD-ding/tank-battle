# 地图格式

## 瓦片类型

| 值 | 类型 | 说明 |
|----|------|------|
| 0 | 空地 | 可通行 |
| 1 | 砖墙 | 可被子弹摧毁，摧毁后变为空地 |
| 2 | 钢墙 | 不可摧毁，子弹命中后消失 |
| 3 | 水面 | 不可通行，子弹可飞越 |
| 4 | 基地 | 不可通行，被击中则游戏结束 |

## 地图尺寸

- 26 列 × 26 行
- 每个瓦片 32×32 像素
- 画布总尺寸：832 × 832 像素

## 地图模板

地图由 `MAP_TEMPLATE` 字符串数组定义，每个字符对应一个瓦片类型：

```
行 0-1:   空地（敌人出生区域）
行 2-9:   上半区障碍
行 10-11: 空地通道
行 12-17: 中部障碍（含水域）
行 18-21: 下半区障碍
行 22-25: 基地保护区域
```

## 基地布局

基地位于底部中间（第 24-25 行，第 12-13 列），由 2×2 的基地瓦片组成。初始化时自动在基地周围生成 U 形砖墙保护：

```
        砖砖砖砖
        砖    砖
        砖 基地 砖
        砖 基地 砖
```

## 出生点

| 角色 | 位置 |
|------|------|
| 敌人 1 | 左上角 (0, 0) |
| 敌人 2 | 上方中间 (12, 0) |
| 敌人 3 | 右上角 (25, 0) |
| 玩家主点 | 底部中间 (12, 21) |
| 玩家备选 1 | 左下 (9, 24) |
| 玩家备选 2 | 右下 (15, 24) |

敌人在三个出生点间轮询，如果当前点被占则尝试下一个。玩家复活时依次尝试三个出生点，均被占则强制在主点复活并给予额外无敌时间。
