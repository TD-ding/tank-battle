# 游戏玩法

## 概述

坦克大战是经典 FC 游戏的 HTML5 Canvas 复刻版。玩家操控坦克在地图上移动、射击，消灭所有敌方坦克即可获胜。底部有需要保护的基地——基地被摧毁则游戏失败。

## 操作方式

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| P / Esc | 暂停 / 继续 |
| M | 静音 / 取消静音 |
| Enter | 开始 / 重新开始 |

## 道具

击毁敌方坦克有 25% 概率掉落道具，地图也会每隔一段时间随机刷新道具（同时场上最多 2 个）。

| 标记 | 效果 | 颜色 | 持续时间 |
|------|------|------|----------|
| S | 玩家移动加速 1.5 倍 | 绿色 | 8 秒 |
| + | 额外 1 条生命（上限 5 条） | 红色 | 即时生效 |
| D | 基地护盾，期间基地不可被摧毁 | 蓝色 | 8 秒 |
| * | 增强火力，同时可发射 4 颗子弹 | 黄色 | 8 秒 |

道具存在 8 秒后自动消失，最后 2 秒会闪烁提示。

## 游戏规则

- 玩家拥有 3 条初始生命，被击中后复活并获 2 秒无敌时间（闪烁效果）
- 共 20 个敌方坦克，场上同时最多 4 个，全部消灭即为胜利
- 消灭一个敌方坦克得 100 分，分数和历史最高分通过 localStorage 持久保存
- 砖墙可被子弹摧毁，钢墙不可摧毁
- 水面区域不可通行，但子弹可以飞越
- 基地被击中则游戏立即结束（护盾保护期间除外）
- 玩家同时最多发射 2 颗子弹（增强火力时为 4 颗），敌方坦克同时最多 1 颗

## 敌人 AI

- 敌人坦克按固定间隔切换移动方向
- 撞墙后立即换方向，并记录已试方向避免反复撞同一面墙
- 敌人越少，追踪玩家的概率越高（30%→80% 线性增长）
- 射击间隔有 ±30% 的随机波动，避免机械感
